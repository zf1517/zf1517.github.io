js数组和字典

我们应当尽量将Object的直接实例作为字典，而不是其子类，这样可以减少原型污染，因为一般我们不会改写Object.prototype。更彻底的方式是一开始就不使用原型。

var x = Object.creat(null);
Object.getPrototypeOf(x) === null; //true

var x = {_proto_:null}
x instanceof Object //false 不推荐

	Array.protoype.forEach.call(arguments,function(item){
