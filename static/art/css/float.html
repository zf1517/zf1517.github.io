float深入理解

1. 概述

float可以改变元素的正常文档流顺序，会使得父元素高度坍陷，随后的元素跟随。利用浮动可以实现复杂布局。

2. 清除浮动

  • clear:both
    • 后面添加一个额外的div，并设置clear：both;
    • 父元素使用伪元素
      	.clearfix:after{ 
	    content: '';
		display: block;						
		clear: both;						
		height: 0;						
		visibility: hidden;													
	}					    
	.clearfix{ /*兼容 IE*/						
		zoom: 1;						
	}
  • 父元素BFC(块状格式化上下文)形成封闭的容器(运用特定场景)
    • 父元素float
    • 父元素position:absolute/fixed
    • 父元素overflow：hidden/scroll
    • 父元素display：inline-block/table-cell
    • 父元素设置width/height/zoom:1

3. 其他

设置浮动的元素会隐式改变其display属性为block。
