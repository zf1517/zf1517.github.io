深入理解border

1.实现半透明边框效果

默认情况下，background区域包含至边框外边沿，为了实现半透明边框，需要将背景切至边框内边沿。可利用background-clip属性实现，该属性默认为boder-box,可设为padding-box和content-box.

	    border:10px solid hsla(0,0%,100%,.5);
	    background:white;			
	    background-clip:padding-box

2.多重边框

当只需要两层边框时，可以用outline属性来实现，优点是可以产生虚线边框，缺点是当设置boder-radius时，描边无法跟随圆角

        border:10px solid #655;
        outline:5px solid deeppink;

当需要多重边框时，可使用box-shadow属性，它支持逗号分隔语法，可以创造任意数量的投影。

        border:10px solid #16EC12;
        box-shadow: 0 0 0 10px #655,
                    0 0 0 15px deeppink,
                    0 2px 5px rgba(0,0,0,.6);
        // h-shadow v-shadow blur spread color inset

3.boder构建三角形

border可为每个方向上设置颜色，当content区域为0时，border沿对角线分割

    	border-width:12px 20px;
    	border-style:solid;
    	border-color:red transparent green yellow;

4.border画圆

通过设置border-radius设置每个圆角的长半径和短半径来形成椭圆和圆形。

    	width: 100px;
    	height: 100px;
    	border-radius: 50%;
    	width: 100px;
    	height: 100px;
    	border-radius: 50% / 100% 100% 0 0;
